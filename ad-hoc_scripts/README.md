# AD-HOC SCRIPTS

### All the scripts make usefully file from snapshots (see FORMAT.md)

## Slices

The script `slice.py` extracts 2D slices from 3D simulation (entire snapshot)

inputs:
- snapshot file
- axis (x, y, z)
- axis value

outputs:
- 2D slice file



## Histogram

The script `histogram.py` generates histograms from snapshot or slice files

inputs:
- snapshot or slice file
- number of bins

outputs:
- histogram file
